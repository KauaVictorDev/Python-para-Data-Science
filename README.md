# 📊 Python para Data Science

Este projeto foi desenvolvido utilizando o **Google Colab** com o objetivo de praticar e aplicar conceitos fundamentais de **Data Science** usando Python.

---

## 🚀 Objetivo

Realizar análise de dados, manipulação de informações e geração de insights a partir de um conjunto de dados, utilizando bibliotecas populares do ecossistema Python.

---

## 🛠️ Tecnologias utilizadas

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook (Google Colab)

---

## 📂 Estrutura do projeto

```
📁 python-data-science/
│
├── Python para Data Science.ipynb
├── README.md
```

---

## 📈 O que foi feito

* Importação e exploração de dados
* Limpeza e tratamento de dados
* Análise exploratória (EDA)
* Visualização de dados com gráficos
* Geração de insights a partir dos dados

---

## ▶️ Como executar o projeto

1. Acesse o notebook pelo Google Colab
2. Execute as células em ordem
3. Verifique os resultados e gráficos gerados

Ou:

1. Faça o download do arquivo `.ipynb`
2. Abra em um ambiente Jupyter Notebook local
